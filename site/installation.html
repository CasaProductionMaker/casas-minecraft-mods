Installation

This will be a short tutorial on installing Minecraft mods for those of you who have never done this yourselves. The tutorial will assume you know about your computer's file systems. If you do, skip to the tutorial. Anyways,

To install the mod, you will have to go to the filepath C:\Users\(your user name)\Appdata\Roaming\.minecraft. Inside, there will probably be nothing, or maybe a mod's old version. Anyhow, you must delete the contents of the the folder and paste in the modrequirements folder. Make sure the 2 .jar files are inside the .minecraft folder and not the modrequirements folder.
